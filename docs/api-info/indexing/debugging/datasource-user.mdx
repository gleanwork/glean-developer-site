Gives various information that would help in debugging issues related to a particular user. For example, it gives information about the user's permissions, groups, memberships, etc.

### Sample Request

<Tabs>
<TabItem value="curl" label="cURL">

```bash snippet=datasource-user/snippet-01.sh
curl -X POST https://customer-be.glean.com/api/index/v1/debug/gleantest/user
  -H 'Authorization : Bearer <token>' \
  -H 'Content-Type : application/json' \
  -d '{
        "email": "user1@example.com"
      }'
```

</TabItem>
<TabItem value="python" label="Python">

```python snippet=datasource-user/snippet-02.py
from glean.api_client import Glean
import os

with Glean(
    api_token=os.getenv("GLEAN_INDEXING_API_TOKEN", ""),
    server_url=os.getenv("GLEAN_SERVER_URL", ""),
) as client:
    try:
        res = client.indexing.people.debug(
            datasource="gleantest",
            email="user1@example.com",
        )
        print(res)
    except Exception as e:
        print(f"Exception when debugging user: {e}")
```

</TabItem>
</Tabs>
