# Platform API Overview

The Glean Platform API exposes Glean's agentic building blocks — agents, skills,
and search — as a RESTful API so you can embed Glean's reasoning and retrieval
directly into your own products and automation.

:::info Experimental
The Platform API is experimental. Endpoints are marked with `x-glean-experimental`
and may change without the usual deprecation notice period. Avoid depending on it
for production-critical workloads until it is generally available.
:::

## Key Capabilities

<CardGroup cols={2}>
<Card title="Agents" icon="Cpu">
  Discover agents, inspect their input/output schemas, and execute agent runs programmatically.
</Card>

<Card title="Skills" icon="Zap">
  List and retrieve the skills available in your instance, including status and sync provenance.
</Card>

<Card title="Search" icon="Search">
  Query across your organization's connected content with filters and time ranges.
</Card>
</CardGroup>

## Developer Workflow

<Steps>
<Step title="Authentication Setup">
Authenticate your requests with a Glean API token. See the
[Authentication Guide](../authentication/overview) for details.
</Step>

<Step title="Explore the Reference">
Browse the [Platform API Reference](/api/platform-api/agents-overview) to see the
available agents, skills, and search endpoints and their request/response shapes.
</Step>

<Step title="Start Building">
Call the endpoints directly over HTTP, or use the request/response examples in the
reference to scaffold your integration.
</Step>
</Steps>
