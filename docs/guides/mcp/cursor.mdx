# Glean for Cursor

Connect Cursor to Glean's remote MCP server to give its AI access to your company's docs, decisions, code, people, and tickets — all respecting your existing access controls.

## Installation

Use the [MCP Configurator](/guides/mcp) to generate your server URL and credentials. Select **Cursor** as your host, enter your Glean instance details, and follow the instructions to add the server in Cursor's settings.

The configurator will walk you through authentication — see [Authentication](/get-started/authentication) for details on the available options.
