# Glean for Claude Code

Connect Claude Code to Glean's remote MCP server to give it access to your company's docs, decisions, code, people, and tickets — all respecting your existing access controls.

## Installation

Use the [MCP Configurator](/guides/mcp) to generate your setup command. Select **Claude Code** as your host, enter your Glean instance details, and follow the instructions.

The configurator will walk you through authentication — see [Authentication](/get-started/authentication) for details on the available options.
