Glean offers official API clients for several popular programming languages to help developers integrate Glean's powerful search and knowledge capabilities into their applications. These clients provide language-specific interfaces to Glean's APIs, making it easier to build custom solutions without having to handle the low-level HTTP requests.

<Info>
  **Before you start**: You'll need a Client API token to use these libraries. See the [Authentication Guide](/api-info/client/authentication/overview) for complete setup instructions.
</Info>

<Warning>
  **Important**: These API clients are designed for the **Client API only**. For the Indexing API, you'll need to use HTTP requests directly with [Indexing API tokens](/api-info/indexing/authentication/overview).
</Warning>

## Quick Start

<Steps>
  <Step title="Get your credentials">
    Before installing any client, you'll need:
    - **Glean instance name** (your-company)
    - **Client API token** ([create one here](/api-info/client/authentication/glean-issued))
    
    Store these as environment variables:
    ```bash
    export GLEAN_INSTANCE="your-company"
    export GLEAN_API_TOKEN="your-token-here"
    ```
  </Step>
  
  <Step title="Install your preferred client">
    Choose your language and follow the installation instructions below.
  </Step>
  
  <Step title="Test your setup">
    Each language section includes a quick test to verify everything is working.
  </Step>
</Steps>

## Installation & Usage

<Tabs>
  <TabItem value="python" label="Python">

<Steps>
      <Step title="Install the Python client">
        ```bash
        pip install glean-api-client
        ```
        
        Or using Poetry:
        ```bash
        poetry add glean-api-client
        ```
      </Step>
      
      <Step title="Basic usage">
        ```python
        from glean.api_client import Glean
        import os

        with Glean(
            api_token=os.getenv("GLEAN_API_TOKEN"),
            instance=os.getenv("GLEAN_INSTANCE"),
        ) as client:
            # Send a chat message
            response = client.client.chat.create(
                messages=[{
                    "fragments": [{"text": "What are our company values?"}]
                }],
                timeout_millis=30000
            )
            print(response)
        ```
      </Step>
    </Steps>

</TabItem>
  
  <TabItem value="typescript" label="TypeScript">

<Steps>
      <Step title="Install the TypeScript client">
        ```bash
        npm install @gleanwork/api-client
        ```
        
        Or using Yarn:
        ```bash
        yarn add @gleanwork/api-client
        ```
      </Step>
      
      <Step title="Basic usage">
        ```typescript
        import { Glean } from "@gleanwork/api-client";

        const client = new Glean({
          apiToken: process.env.GLEAN_API_TOKEN,
          instance: process.env.GLEAN_INSTANCE,
        });

        const result = await client.client.chat.create({
          messages: [{
            fragments: [{ text: "What are our company values?" }]
          }]
        });

        console.log(result);
        ```
      </Step>
    </Steps>

</TabItem>
  
  <TabItem value="go" label="Go">

<Steps>
      <Step title="Install the Go client">
        ```bash
        go get github.com/gleanwork/api-client-go
        ```
      </Step>
      
      <Step title="Basic usage">
        ```go
        package main

        import (
          "context"
          "fmt"
          "os"
          glean "github.com/gleanwork/api-client-go"
        )

        func main() {
          ctx := context.Background()

          client := glean.New(
            glean.WithAPIToken(os.Getenv("GLEAN_API_TOKEN")),
            glean.WithInstance(os.Getenv("GLEAN_INSTANCE")),
          )

          message := glean.ChatMessageFragment{
            Text: "What are our company values?",
          }

          res, err := client.Client.Chat.Create(ctx, &glean.ChatRequest{
            Messages: []glean.ChatMessage{{
              Fragments: []glean.ChatMessageFragment{message},
            }},
            TimeoutMillis: glean.Int64(30000),
          })

          if err != nil {
            fmt.Printf("Error: %v\n", err)
            return
          }

          fmt.Println(res)
        }
        ```
      </Step>
    </Steps>

</TabItem>
  
  <TabItem value="java" label="Java">

<Steps>
      <Step title="Add dependency">
        **Maven:**
        ```xml
        <dependency>
          <groupId>com.glean.api-client</groupId>
          <artifactId>glean-api-client</artifactId>
          <version>0.x.x</version>
        </dependency>
        ```
        
        **Gradle:**
        ```groovy
        implementation 'com.glean.api-client:glean-api-client:0.x.x'
        ```
      </Step>
      
      <Step title="Basic usage">
        ```java
        import com.glean.api_client.glean_api_client.Glean;
        import com.glean.api_client.glean_api_client.models.components.*;
        import java.util.List;

        public class Main {
          public static void main(String[] args) {
            Glean client = Glean.builder()
                .apiToken(System.getenv("GLEAN_API_TOKEN"))
                .instance(System.getenv("GLEAN_INSTANCE"))
                .build();

            var response = client.client().chat().create()
                .chatRequest(ChatRequest.builder()
                    .messages(List.of(
                        ChatMessage.builder()
                            .fragments(List.of(
                                ChatMessageFragment.builder()
                                    .text("What are our company values?")
                                    .build()))
                            .build()))
                    .build())
                .call();

            if (response.chatResponse().isPresent()) {
              System.out.println(response.chatResponse().get());
            }
          }
        }
        ```
      </Step>
    </Steps>

</TabItem>

## Authentication with API Clients (Client API Only)

The API clients support both Client API authentication methods:

<Warning>
  **OAuth limitation**: OAuth authentication only works with Client API endpoints. These API clients are designed for Client API only, so OAuth will work. For Indexing API operations, use direct HTTP requests with [Indexing API tokens](/api-info/indexing/authentication/overview).
</Warning>

### Using User-Scoped Tokens (Recommended)
```python
# Most common - token is tied to a specific user
client = Glean(
    api_token="your-user-token",
    instance="your-company"
)
```

### Using Global Tokens
```python
# For multi-user applications - requires X-Glean-ActAs header
client = Glean(
    api_token="your-global-token",
    instance="your-company"
)

# When making requests, specify the user
response = client.client.chat.create(
    messages=[{"fragments": [{"text": "Hello"}]}],
    headers={"X-Glean-ActAs": "user@company.com"}
)
```

### Using OAuth (Client API Only)
```python
# OAuth tokens work with API clients since they target Client API
client = Glean(
    api_token="your-oauth-token",  # This should be your OAuth access token
    instance="your-company"
)

# Note: You may need to handle the X-Glean-Auth-Type header separately
# depending on the client library implementation
```

## Error Handling

All clients provide structured error handling:

<Tabs>
<TabItem value="python" label="Python">

```python
try:
    response = client.client.chat.create(messages=[...])
except Exception as e:
    print(f"API Error: {e}")
```

</TabItem>
<TabItem value="typescript" label="TypeScript">

```typescript
try {
  const response = await client.client.chat.create({...});
} catch (error) {
  console.error('API Error:', error);
}
```

</TabItem>
<TabItem value="go" label="Go">

```go
response, err := client.Client.Chat.Create(ctx, request)
if err != nil {
    fmt.Printf("API Error: %v\n", err)
}
```

</TabItem>
<TabItem value="java" label="Java">

```java
try {
    var response = client.client().chat().create()...call();
} catch (Exception e) {
    System.err.println("API Error: " + e.getMessage());
}
```

</TabItem>
</Tabs>

## Advanced Features

### Rate Limiting
All clients automatically handle rate limiting with exponential backoff.

### Timeouts
Configure request timeouts for your use case:

<Tabs>
<TabItem value="python" label="Python">

```python
response = client.client.chat.create(
    messages=[...],
    timeout_millis=60000  # 60 seconds
)
```

</TabItem>
<TabItem value="typescript" label="TypeScript">

```typescript
const response = await client.client.chat.create({
  messages: [...],
  timeoutMillis: 60000
});
```

</TabItem>
</Tabs>

### Streaming Responses
For real-time chat responses:

<Tabs>
<TabItem value="python" label="Python">

```python
for chunk in client.client.chat.stream(messages=[...]):
    print(chunk)
```

</TabItem>
<TabItem value="typescript" label="TypeScript">

```typescript
const stream = client.client.chat.stream({messages: [...]});
for await (const chunk of stream) {
  console.log(chunk);
}
```

</TabItem>
</Tabs>

## API Coverage

These API clients cover **Client API endpoints only**:

<Accordion title="Supported Client API Endpoints">

**Endpoints available through API clients:**

- **Chat**: `/rest/api/v1/chat`
- **Search**: `/rest/api/v1/search`  
- **Agents**: `/rest/api/v1/agents/*`
- **Collections**: `/rest/api/v1/collections/*`
- **Answers**: `/rest/api/v1/answers/*`
- **Announcements**: `/rest/api/v1/announcements/*`
- **Authentication**: `/rest/api/v1/createauth`
- All other Client API endpoints

</Accordion>

<Accordion title="Not Supported - Indexing API">

**These endpoints require direct HTTP requests:**

- **Document Indexing**: `/api/index/v1/indexdocument`
- **Datasource Management**: `/api/index/v1/adddatasource`
- **Bulk Operations**: `/api/index/v1/bulkindexdocuments`
- All other Indexing API endpoints

For these operations, use:
- [Indexing API Authentication Guide](/api-info/indexing/authentication/overview)
- [Indexing API Getting Started](/api-info/indexing/getting-started/setup-datasource)
- Direct HTTP requests with appropriate Indexing API tokens

</Accordion>
